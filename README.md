## 설명

LH 청약 공고문 확인 절차가 복잡하여 부모님이 불편함을 느끼심 

때문에 이 프로젝트를 시작.

해당지역에 관한 공고문을 바로 확인할 수 있도록 개발함.

표와 지도로 공고문 정보를 보여주고, 클릭 시 바로 LH 원문 공고문으로 이동
